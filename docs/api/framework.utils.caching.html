class framework.utils.caching.redis_pool.RedisPoolConnection(**kwargs)[source]¶

Bases: object

connection pool to manage connections to a Redis server. By default, each Redis instance you create will in turn create its own connection pool. You can override this behavior and use an existing connection pool by passing an already created connection pool instance to the connection_pool argument of the Redis class.

By default redis open 16 db’s while creating connection. we can specify specific db number to open it.

create redis connection with pooling

Keyword Arguments:
 
  • host (str) – hostname for redis
  • port (str) – port for redis
  • password (str) – password , Default None
  • db_number (int) – db number from 0-15
  • max_connections (int) – maximum number of connections for pooling
